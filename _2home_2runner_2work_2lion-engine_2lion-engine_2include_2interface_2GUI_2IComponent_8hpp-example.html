The IComponent interface

This class is an interface that is used to create a component object by inheriting from it.

The component is represented by 4 values: id, x, y, width and height. The id is a size_t that is unique to each component. The x and y are the coordinates of the component. The width and height are the size of the component.

LE::IComponent component;
component.setPos(42, 42);
