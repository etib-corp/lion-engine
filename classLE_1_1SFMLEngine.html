Detailed Description

Creates an instance of the SFMLEngine class.

Returns
A shared pointer to the created SFMLEngine instance.

The SFMLEngine class is an implementation of the IEngine interface using the SFML library.

The SFMLEngine class provides methods to create various engine components such as fonts, shaders, and windows using the SFML library.
