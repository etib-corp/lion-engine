The ITextField interface.

The ITextField interface

This class is an interface that is used to create a text field object by inheriting from it.

The text field is represented by 3 values: placeholder, label and input. The placeholder is a std::string that contains the placeholder of the text field. The label is a std::shared_ptr<LE::GUI::IText> that contains the label of the text field. The input is a std::shared_ptr<LE::GUI::IText> that contains the input of the text field.

LE::GUI::ITextField textField;
textField.setPlaceholder("Enter your name");
textField.setLabel(std::make_shared<LE::GUI::IText>());
textField.setInput(std::make_shared<LE::GUI::IText>());
