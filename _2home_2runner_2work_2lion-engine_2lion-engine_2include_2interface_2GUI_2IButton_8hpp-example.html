The IButton interface.

The IButton interface

This class is an interface that is used to create a button object by inheriting from it.

The button is represented by 3 values: onClick, onHover and onUnhover. Each value is a std::function<void()> that contains the function to call when the button is clicked, hovered or unhovered.

LE::IButton button;
button.setOnClickCallback([]() {
std::cout << "Button clicked" << std::endl;
});
button.setOnHoverCallback([]() {
std::cout << "Button hovered" << std::endl;
});
button.setOnUnhoverCallback([]() {
std::cout << "Button unhovered" << std::endl;
});
